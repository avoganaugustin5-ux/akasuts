# 🎓 AKASUTS - Université Thomas SANKARA

**AKASUTS** (Accompagnement et Kit d’Orientation de l’Université Thomas SANKARA) est une application mobile moderne conçue pour faciliter l'orientation et l'inscription des étudiants. Elle centralise les informations essentielles pour offrir un parcours académique fluide et numérique.

---

## 🚀 Fonctionnalités Clés
* **Orientation Intelligente** : Explorez les filières (Informatique, Droit, Économie, Physique) et leurs conditions d'admission spécifiques.
* **Guide d'Inscription** : Un parcours balisé pour réussir vos démarches administratives sans erreur.
* **Mode Hors-ligne** : Accédez aux informations cruciales même sans connexion internet stable.
* **Paiement Intégré** : Interface dédiée pour simplifier les frais de scolarité.
* **Ressources PDF** : Bibliothèque de documents officiels pour accompagner votre réussite.

## 🛠️ Stack Technique
Ce projet a été développé avec des technologies de pointe pour garantir performance et évolutivité :
* **Framework** : [Flutter](https://flutter.dev) (Multiplateforme Android/Web)
* **Langage** : Dart & Groovy (pour la configuration Gradle)
* **Backend** : [Firebase](https://firebase.google.com) (Authentification, Cloud Firestore, Storage)
* **CI/CD** : GitHub Actions pour le déploiement automatique sur GitHub Pages.
